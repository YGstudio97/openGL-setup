# 🚀 ZeroSetupGL

> Modern OpenGL starter project for C++ developers.
> No complicated setup. Just clone, code, and run.

This repository is made for beginners and developers who want to start OpenGL instantly without wasting time configuring libraries.

---

# ✨ Features

* ✅ OpenGL preconfigured
* ✅ GLFW included
* ✅ GLAD included
* ✅ Ready-to-run project
* ✅ Beginner friendly
* ✅ Fast setup
* ✅ Clean project structure
* ✅ No extra configuration needed
